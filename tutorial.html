How to use Our Nikli-24 Copyrighter

Go To App / Use Software

Step 1: Select A Photo

Select the photo you want to set a watermark on.

Step 2: Set Opacity

Set how much the logo / watermark should be visible by selecting a proper opacity.

Step 3: Set Logo Size

Set your desired logo / watermark size.

Step 4: Select Logo Position

Select in which position you want to set the logo / watermark.

Step 5: Extra Feature

If the logo / watermark properties doesn't change as per your setting, click this button after changing any property.

Step 6: Save the photo

Save the edited photo to your device by clicking this button. After clicking download button, wait for 5-10 seconds as the full quality image may take some time as per your devices rendering power.

Step 7: Additional Step for Download Error

Click on this "Click Here" link if the download has not started within 20 seconds after clicking the download button.
